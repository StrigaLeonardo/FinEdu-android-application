# Inicijalne upute za prijavu projekta iz Razvoja mobilnih aplikacija i igara

# FinEdu

## Opis domene

FinEdu je edukativna mobilna aplikacija za studente koja omogućuje učenje o ulaganju, burzi, kriptovalutama i osobnim financijama kroz interaktivne module, kvizove i simulaciju ulaganja na virtualnoj burzi. Korisnici mogu ulagati virtualni novac, pratiti i analizirati svoj portfelj te kroz kvizove provjeravati i nadograđivati znanje. Sve funkcionalnosti osmišljene su kako bi korisnici stekli praktične vještine upravljanja novcem i investiranjem, bez rizika od stvarnog financijskog gubitka.

## Specifikacija projekta

Sve funkcionalnosti su dizajnirane tako da korisnicima pruže praktično iskustvo i znanje iz financija bez rizika od stvarnog gubitka novca.

| Oznaka | Naziv                        | Kratki opis                                                                                                                                    |
| ------ | ---------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| F01    | Simulacija ulaganja          | Korisnik može ulagati virtualni novac u dionice, fondove ili kriptovalute, pratiti promjene vrijednosti i donositi odluke kao na pravoj burzi. |
| F02    | Edukativni moduli i kvizovi  | Interaktivni edukativni sadržaji i kvizovi iz područja osobnih financija, ulaganja, štednje, rizika i kriptovaluta.                            |
| F03    | Praćenje i analiza portfelja | Prikaz vrijednosti virtualnog portfelja kroz vrijeme, analiza uspješnosti ulaganja, vizualizacija rezultata i preporuke za poboljšanje.        |

## Tehnologije i oprema

Android Studio (Kotlin) – razvoj mobilne aplikacije

Firebase – autentikacija korisnika, baza podataka, pohrana rezultata i portfelja

REST API (npr. za dohvat tržišnih podataka)

MPAndroidChart ili slična biblioteka – za vizualizaciju portfelja i rezultata

Git i GitHub – verzioniranje koda

GitHub Wiki – tehnička i projektna dokumentacija

GitHub Projects – planiranje i praćenje zadataka

Jira - upravljanje projektom
